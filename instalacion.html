Guia de instalacion

  1. Descomprimir srv2.0 en la carpeta raiz de tu servidor.

  2. lanzar el scrip de creacion de la base de datros que se encuentra en la carpeta install, con nombre creacion.sql.

  3. si tienes un archivo sql salvado de tu servicio importalo despues.
    si es la primera vez que entras tienes un archivo llamado servicios.sql de la misma carpeta ,con tecnicos, clientes y servicios de prueba que podras modificarlo en tu base de datos.

  4. en el caso de alojar la base de datos en otra ruta que no sea localhost tendras k modificar el archivo config/conexion.php
    y tendras que crear un usuario de nombre select con permisos de acceso externos.

1º entrada

  • el archivo sql servicios.sql incluye 2 usuarios:
    - administrador user:admin pass:admin
    - comun user:user pass:user

    puedes agregar mas en la base de datos, con nivel 1 para admin y 0 para current user .

  • como clientes de prueva tienes cliente1, cliente2 etc , puedes agregar mas desde la ventana clientes o desde la base de datos
