Guia de instalacion

  1. Descomprimir srv2.0 en la carpeta raiz de tu servidor.

  2. lanzar el scrip de creacion de la base de datros que se encuentra en la carpeta instalacion, con nombre creacion.sql.

  3. si tienes un archivo sql salvado de tu servicio importalo despues.
    si es la primera vez que entras tienes un archivo llamado servicios.sql con tecnicos, clientes y servicios de prueba que podras modificarlo en tu base de datos.

  4. en el caso de alojar la base de datos en otra ruta que no sea localhost tendras k modificar el archivo configuracion/conexion.php
    y tendras que crear un usuario de nombre select con permisos de acceso externos.
